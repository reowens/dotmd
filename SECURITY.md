# Security Policy

## Supported Versions

Only the latest published version of `dotmd-cli` receives security updates.

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub's private vulnerability reporting](https://github.com/reowens/dotmd/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

You should expect an initial response within 7 days. If the vulnerability is confirmed, a fix will be released as soon as practical.
